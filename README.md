# Vue Task App
From the tutorial: https://www.youtube.com/watch?v=qZXt1Aom3Cs

## Project setup
```
npm install
```

## Run the JSON-Server backend (http://localhost:5000)
```
npm run backend
```

### Run the vue development server (http://localhost:8080)
```
npm run serve
```

### Compile and minify for production
```
npm run build
```
